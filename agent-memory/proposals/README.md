# Global memory proposals (staging)

Unpromoted candidate learnings for the **global** tier. Agents/sessions drop a
`memory_proposal` here (or emit it in their return); the architect / `memory-curator`
reviews via the `memory-curate` skill and promotes keepers into
`~/.claude/agent-memory/<role>/MEMORY.md`.

Promote only durable, evidence-backed learnings. Reject vague, secret,
customer-specific, one-off, or unsupported entries. This is staging, not memory.
